###Styling

Global style - index.css and app.css are global files. Classes defined in Any jsx file can be defined/styled in either of these .css files, and it will be applicable to all the .jsx files available in the source folder. 

External style - Each jsx file having it's own .css file, imported in the respective component file. Priority?

Module style - ModuleName.module.css and import XYZ from "./address" and then use className = {XYZ.whateverClassNameYouDefinedInTheFile}

Inline style - declare style = {{stylingWithCamelCase}}

Object style - create styling as an object, then pass it as style = {styleObj}
